# Stratos n8n Automation Library

This folder turns Stratos from “premium website + AI idea” into an implementation-ready automation offer.

## How to use this library

1. Pick the workflow that matches the prospect's clearest leak.
2. Read the guide before editing nodes.
3. Import the matching `.workflow.json` into n8n.
4. Replace every placeholder credential/value with the client's real systems.
5. Test with dummy data first, then with one real internal lead.
6. Screenshot the successful test and add it to the Proof Ledger.

## Golden rule

Do **not** sell vague automations. Sell a specific trigger → routing → notification → follow-up → proof loop.

## Workflow templates included

- `templates/medspa-consult-capture.workflow.json`
- `templates/missed-call-text-back.workflow.json`
- `templates/review-request-engine.workflow.json`
- `templates/lead-to-close-room-router.workflow.json`

All credentials are placeholders. No client secrets belong in the repo.
