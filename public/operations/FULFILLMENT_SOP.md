# Stratos Fulfillment SOP

## Phase 1 — Diagnose
- Complete client intake snapshot.
- Identify the highest-value leak: first impression, lead capture, missed calls, follow-up, proof, or reporting.
- Pick one primary automation first.

## Phase 2 — Build
- Build the conversion page/site path.
- Configure the n8n workflow with placeholder-free test data.
- Add staff notification and lead log.
- Add fallback/error path.

## Phase 3 — Prove
- Test mobile CTA → webhook → staff alert → log.
- Screenshot test proof.
- Add QA notes and limitations.
- Get client approval before launch.

## Phase 4 — Retain
- Send weekly owner summary.
- Identify next bottleneck.
- Add one optimization per month.
