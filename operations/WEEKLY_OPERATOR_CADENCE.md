# Weekly Operator Cadence

## Monday: Pipeline command
- Pick 10 targets.
- Send 10 precise first touches.
- Build or refresh 2 close rooms.
- Log status and next action.

## Wednesday: Delivery/proof command
- QA active sites/automations.
- Capture before/after screenshots.
- Add one proof-ledger entry.
- Convert one demo into a transparent case-study block.

## Friday: Retention command
- Send owner summary to clients/prospects in active delivery.
- Report leads captured, missed calls recovered, review requests sent, and bottlenecks.
- Recommend one next optimization.

## Never skip
- No unsupported claims.
- No vague “AI automation” deliverables.
- No work leaves the repo without validation/export.
